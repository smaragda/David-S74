# David-S74

Staticky web pro GitHub Pages.

## Pravidla repozitare

- `main` obsahuje jen deployovatelny web a soubory potrebne pro GitHub Pages.
- Archivni a pracovni materialy patri do vedlejsich branchek, ne do `main`.
- GitHub Pages se nasazuji pouze z `main`.

## Produkcni obsah v `main`

- `index.html`
- `css/`
- `js/`
- `data/`
- runtime pouzivane soubory v `images/`
- `.github/workflows/deploy-pages.yml`
